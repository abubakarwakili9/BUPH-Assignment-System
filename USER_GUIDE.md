# User Guide - Online Assignment Submission System

## Welcome!

This guide will help you understand how to use the Online Assignment Submission System.

---

## Getting Started

### First Time Setup
1. Go to the application URL
2. Click "Register" to create an account
3. Choose your role (Student or Instructor)
4. Fill in your details and submit

---

## For Students

### Dashboard Overview
When you login, you'll see:
- List of all assignments
- Assignment status (Not Submitted, Submitted, Graded)
- Due dates and grades

### How to Submit an Assignment

1. **Find the assignment** on your dashboard
2. **Click "Submit Assignment"** button
3. **Read the assignment details**:
   - Assignment title
   - Course name
   - Due date
   - Description
   - Total marks

4. **Upload your file**:
   - Click "Choose File"
   - Select your assignment file
   - Supported formats: PDF, DOC, DOCX, TXT, ZIP
   - Maximum size: 16MB

5. **Click "Upload Assignment"**
6. You'll see a success message

### After Submission
- You can see your submission status on the dashboard
- Once graded, you'll see:
  - Your grade
  - Instructor's feedback

### Resubmitting an Assignment
1. Find the assignment on your dashboard
2. Click "Resubmit" button
3. Upload new file
4. Submit

---

## For Instructors

### Dashboard Overview
When you login, you'll see:
- Your assigned courses
- List of assignments you've created
- Number of submissions for each assignment

### Creating an Assignment

1. **Click "Create New Assignment"** button
2. **Fill in the details**:
   - **Title**: Name of the assignment (e.g., "Final Project Report")
   - **Course**: Select from your assigned courses
   - **Description**: Provide clear instructions for students
   - **Due Date**: Set deadline for submission
   - **Total Marks**: Maximum points (default: 100)

3. **Click "Create Assignment"**

### Viewing Submissions

1. **Click "View Submissions"** on any assignment
2. You'll see a table with:
   - Student name and ID
   - Submission date/time
   - Current status
   - Grade (if graded)

### Downloading Student Work

1. Find the student's submission
2. Click "Download" button
3. Review the file on your computer

### Grading an Assignment

1. **Click "Grade"** button for a student's submission
2. **Enter the grade**:
   - Must be between 0 and total marks
   - Example: For 100 marks assignment, enter between 0-100

3. **Provide feedback**:
   - Write detailed comments
   - Explain strengths and areas for improvement
   - Suggest corrections if needed

4. **Click "Save Grade"**
5. Student will see grade and feedback on their dashboard

### Editing a Grade
You can update grades anytime:
1. Go to submissions
2. Click "Edit Grade"
3. Update grade and feedback
4. Save changes

---

## For Administrators

### Dashboard Overview
You'll see:
- System statistics (total students, instructors, courses, etc.)
- Recent users
- Recent courses

### Creating a Course

1. **Click "Create New Course"** button
2. **Fill in details**:
   - **Course Code**: Unique identifier (e.g., CS101)
   - **Course Name**: Full course name
   - **Department**: Course department
   - **Instructor**: Assign an instructor (optional)

3. **Click "Create Course"**

### Managing Users

View all users from the dashboard:
- Students: Can submit assignments
- Instructors: Can create assignments and grade
- Admins: Full system access

## Common Questions

### Q: What file types can I submit?
**A:** PDF, DOC, DOCX, TXT, and ZIP files up to 16MB.

### Q: Can I submit an assignment after the due date?
**A:** Yes, the system allows late submissions, but your instructor will see when it was submitted.

### Q: Can I delete a submission?
**A:** No, but you can resubmit which replaces the previous file.

### Q: How do I know if my assignment was graded?
**A:** Check your dashboard - graded assignments show your score and have instructor feedback.

### Q: Can instructors download multiple submissions at once?
**A:** Currently, files must be downloaded individually.

### Q: What if I can't see my course?
**A:** Contact your administrator - they need to create the course and assign your instructor.

### Q: How do I change my password?
**A:** Currently, contact your administrator to reset your password.

### Q: What if the file upload fails?
**A:** Check:
- File size is under 16MB
- File type is supported
- You have stable internet connection

## Tips & Tricks

### Students
- **Organize your files**: Name files clearly (e.g., "StudentID_Assignment1.pdf")
- **Check format**: Make sure your file is in an accepted format
- **Test upload**: Try uploading a small test file first if you're unsure

### Instructors
- **Be specific**: Clear assignment descriptions reduce student confusion
- **Use rubrics**: Include grading criteria in descriptions
- **Grade consistently**: Apply same standards to all students
- **Provide examples**: Show what you expect if possible
